# SimpleEngine
A WIP 2d/3d rendering engine for use in my computer graphics endeavors.

## Features
* Line drawing
* Circle drawing
* 2D transformations (translate, rotate, scale)
* 3D transformations
* Load and save models
* Perspective projection
* Viewport and camera manipulation

# SimpleEngine
A WIP 2d/3d engine for use in my computer graphics endeavors.

## Features
* Line drawing
* Circle drawing
* 2d transformations (translate, rotate, scale)
* Load and save from file
* 3d -> 2d perspective projection
* Viewport and camera settings
